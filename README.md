# ocean_toolbox

A personal Python toolbox for oceanographic data processing and analysis, developed alongside active research into sea ice variability, ocean stratification, and polar hydrography across the Southern Ocean and Arctic.

Covers data processing and filtering, physical oceanography calculations (mixed layer depth, grid cell area), spatial statistics (area-weighted means, correlations with autocorrelation-adjusted significance), gridding utilities, and plotting tools for publication-quality figures including circumpolar maps and multi-panel time series.

Built primarily around `xarray`, `numpy`, `matplotlib`, `cartopy`, `cmocean`, `gsw` (TEOS-10), and `scipy`. Intended as a living codebase — functions are added and refined as new projects demand.
